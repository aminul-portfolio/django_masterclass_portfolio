Python Basics: Numbers, Strings, and Lists

Explore the foundational elements of Python including numbers, string operations, and lists with real-world examples.

1️⃣ Basic Arithmetic
print(2 + 3)
print(10 / 2)
  • 🔹 Function Used: Arithmetic operators: +, /
  • 📘 What You Learn: Python can act like a basic calculator.
  • ♻️ Reusability: Used in financial apps, statistics, and AI calculations.
  • 💬 Inline Comments: Addition and division between integers.
2️⃣ Variable Assignment
a = 5
a = a + 3
print(a)
  • 🔹 Function Used: Variable assignment and arithmetic
  • 📘 What You Learn: Reassigning variables using existing values.
  • ♻️ Reusability: Common in loops, counters, and accumulation logic.
  • 💬 Inline Comments: Assign and update a variable with new value.
3️⃣ String Creation and Print
print("Hello, World!")
print('Python is fun.')
  • 🔹 Function Used: print()
  • 📘 What You Learn: Create strings with quotes and print them.
  • ♻️ Reusability: Used in logs, UI messages, and formatting outputs.
  • 💬 Inline Comments: Strings can be defined with single or double quotes.
4️⃣ String Indexing
s = "Python"
print(s[0])
print(s[-1])
  • 🔹 Function Used: Indexing with []
  • 📘 What You Learn: Access characters by position from a string.
  • ♻️ Reusability: Needed in parsing, validation, and substring checks.
  • 💬 Inline Comments: Use indexes to grab letters from a string.
5️⃣ Creating a List
my_list = [1, 2, 3, 4]
print(len(my_list))
  • 🔹 Function Used: len(), list brackets []
  • 📘 What You Learn: Create lists and measure their length.
  • ♻️ Reusability: Lists are core to data analysis, loops, and APIs.
  • 💬 Inline Comments: Store multiple values and get their count.
6️⃣ List Append and Pop
my_list = [10, 20]
my_list.append(30)
my_list.pop()
print(my_list)
  • 🔹 Function Used: .append(), .pop()
  • 📘 What You Learn: Modify lists by adding and removing items.
  • ♻️ Reusability: Useful in queues, stacks, and dynamic lists.
  • 💬 Inline Comments: Append adds to end, pop removes last.
