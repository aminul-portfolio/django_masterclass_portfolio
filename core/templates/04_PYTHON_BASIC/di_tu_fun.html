Professional Use of Dictionaries, Tuples, and Functions

This module combines practical examples of Python dictionaries, tuple operations, and functions to solve industry-level tasks.

1️⃣ Creating and Accessing a Dictionary

my_dict = {'key1': 'value1', 'key2': 'value2'}
print(my_dict['key2'])
  • 🔹 Function Used: dict()
  • 📘 What You Learn: Basic dictionary creation and key access
  • ♻️ Reusability: Common in config setup, key-value APIs
  • 💬 Inline Comments: Defines key-value pairs, accesses value via key
  • 📎 Related Topics: JSON, config files, NoSQL
2️⃣ Nested Dictionary Access

d = {'key1': {'nestkey': {'subnestkey': 'value'}}}
print(d['key1']['nestkey']['subnestkey'])
  • 🔹 Function Used: dict nesting
  • 📘 What You Learn: Accessing deeply nested values
  • ♻️ Reusability: Useful in JSON APIs and hierarchical data
  • 💬 Inline Comments: Traverses multiple levels of nested dictionaries
  • 📎 Related Topics: JSON parsing, MongoDB
3️⃣ Dictionary with Lists and Methods

my_dict = {'key1': 123, 'key2': [12, 23, 33], 'key3': ['item0', 'item1']}
print(my_dict['key3'][0].upper())
  • 🔹 Function Used: upper()
  • 📘 What You Learn: Combining data structures and string methods
  • ♻️ Reusability: Dynamic access and processing of API results
  • 💬 Inline Comments: Access nested list element and apply string method
  • 📎 Related Topics: NLP, web scraping
4️⃣ Dictionary Key Assignment

d = {}
d['animal'] = 'Dog'
d['answer'] = 42
print(d)
  • 🔹 Function Used: dict assignment
  • 📘 What You Learn: Dynamically create dictionary entries
  • ♻️ Reusability: Populating objects in real-time apps
  • 💬 Inline Comments: Adds keys and values on the fly
  • 📎 Related Topics: session state, data pipelines
5️⃣ Dictionary Keys and Values

d = {'key1': 1, 'key2': 2, 'key3': 3}
print(d.keys())
print(d.values())
  • 🔹 Function Used: keys(), values()
  • 📘 What You Learn: Retrieve keys/values from a dict
  • ♻️ Reusability: Parsing and mapping dictionary contents
  • 💬 Inline Comments: Returns dict views of keys and values
  • 📎 Related Topics: iteration, list generation
6️⃣ Tuple Unpacking in Loops

pairs = [(1, 'a'), (2, 'b'), (3, 'c')]
for num, char in pairs:
    print(num, char)
  • 🔹 Function Used: tuple unpacking
  • 📘 What You Learn: Clean iteration over paired data
  • ♻️ Reusability: Useful in data pipelines, CSV parsing
  • 💬 Inline Comments: Automatically separates tuple values in loop
  • 📎 Related Topics: zip, pandas iterrows()
7️⃣ Tuple Index Access

coordinates = (45.4215, -75.6995)
print("Latitude:", coordinates[0])
  • 🔹 Function Used: Tuple indexing
  • 📘 What You Learn: Access values in a tuple
  • ♻️ Reusability: Geolocation, coordinate systems
  • 💬 Inline Comments: Retrieves first item from a fixed-size tuple
  • 📎 Related Topics: GPS APIs, data coordinates
8️⃣ Dictionary Update Method

info = {'name': 'Alice'}
info.update({'age': 30})
print(info)
  • 🔹 Function Used: update()
  • 📘 What You Learn: How to modify dict data efficiently
  • ♻️ Reusability: Updating dynamic user or config data
  • 💬 Inline Comments: Merges new key-value pair into existing dict
  • 📎 Related Topics: patch requests, mutable data
9️⃣ Function Returning Boolean

def is_even(n):
    return n % 2 == 0

print(is_even(8))
  • 🔹 Function Used: return
  • 📘 What You Learn: Create boolean-returning logic
  • ♻️ Reusability: Filtering, validation pipelines
  • 💬 Inline Comments: Returns True only if number is divisible by 2
  • 📎 Related Topics: Lambda, filters, testing
🔟 Using *args in a Function

def add_all(*nums):
    return sum(nums)

print(add_all(1, 2, 3, 4))
  • 🔹 Function Used: *args
  • 📘 What You Learn: Accept variable number of arguments
  • ♻️ Reusability: Useful in financial calculations and logging
  • 💬 Inline Comments: Passes all values as tuple to the function
  • 📎 Related Topics: decorators, logging, unpacking
1️⃣1️⃣ Dictionary with List Comprehension

names = ['Anna', 'Bob', 'Cara']
lengths = {name: len(name) for name in names}
print(lengths)
  • 🔹 Function Used: Dictionary comprehension
  • 📘 What You Learn: Build dictionaries dynamically
  • ♻️ Reusability: Report generation, API summarization
  • 💬 Inline Comments: Maps each name to its character count
  • 📎 Related Topics: comprehension, pandas, functional programming
1️⃣2️⃣ Tuple as Dictionary Key

location = {(40.7128, -74.0060): "New York"}
print(location[(40.7128, -74.0060)])
  • 🔹 Function Used: Tuple as immutable key
  • 📘 What You Learn: Use tuples in lookup operations
  • ♻️ Reusability: GIS systems, financial tickers
  • 💬 Inline Comments: Tuples can serve as unique keys in dictionaries
  • 📎 Related Topics: database keys, caching
