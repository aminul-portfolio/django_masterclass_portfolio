Python Advanced: OOP Examples

Explore real-world Object-Oriented Programming use cases using Python classes, inheritance, methods, and special functions.

Basic Class Initialization (Agent)
            
            class Agent:
                def __init__(self, real_name):
                    self.real_name = real_name

            agent = Agent("Alice")
            print(agent.real_name)
  • 🔹 Function/Class Used: class, __init__, self
  • 📘 What You Learn: Basic class creation, instance initialization
  • ♻️ Reusability: Foundation for creating custom object types
  • 💬 Inline Comments: Uses constructor method to define attributes
  • 📌 Related Topics: Object creation, instantiation
Circle Class with Area Calculation
            
            class Circle:
                pi = 3.14

                def __init__(self, radius=1):
                    self.radius = radius

                def area(self):
                    return self.radius ** 2 * Circle.pi
  • 🔹 Function/Class Used: class, __init__, area()
  • 📘 What You Learn: Class attributes vs instance attributes
  • ♻️ Reusability: Adaptable to any shape calculation or formula engine
  • 💬 Inline Comments: Demonstrates method and attribute structure
  • 📌 Related Topics: Class method, geometric formulas
Readable Object Output with __str__
            
            class Book:
                def __init__(self, title, author):
                    self.title = title
                    self.author = author

                def __str__(self):
                    return f"{self.title} by {self.author}"
  • 🔹 Function/Class Used: __str__
  • 📘 What You Learn: String representation of objects
  • ♻️ Reusability: Printable user-friendly outputs for class instances
  • 💬 Inline Comments: Makes objects readable when printed
  • 📌 Related Topics: Magic methods
Shared Class Attribute Example (Dog)

            class Dog:

  • 📘 What You Learn: Shared class attributes

Employee Bonus Method

            class Employee:
                def __init__(self, name, salary):
                    self.name = name
                    self.salary = salary

                def bonus(self):
                    return self.salary * 0.10
  • 🔹 Function/Class Used: __init__, bonus()
  • 📘 What You Learn: Encapsulation and basic behavior
  • ♻️ Reusability: Can be extended to payroll systems
  • 💬 Inline Comments: Business logic inside methods
  • 📌 Related Topics: Encapsulation, salary management
Bank Account with Deposit Method

                class BankAccount:
                    def __init__(self, balance=0):
                        self.balance = balance

                    def deposit(self, amount):
                        self.balance += amount
  • 🔹 Function/Class Used: __init__, deposit()
  • 📘 What You Learn: Mutating internal state
  • ♻️ Reusability: Great for banking apps or wallets
  • 💬 Inline Comments: Tracks and updates balance
  • 📌 Related Topics: State management, transactions
Static Method for Utility Calculation

            class Math:
                @staticmethod
                def add(x, y):
                    return x + y
  • 🔹 Function/Class Used: @staticmethod
  • 📘 What You Learn: Static methods don't access instance data
  • ♻️ Reusability: Useful for utility or helper classes
  • 💬 Inline Comments: Acts like a function inside class
  • 📌 Related Topics: Decorators, class utilities
