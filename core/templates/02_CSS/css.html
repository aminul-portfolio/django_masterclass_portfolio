CSS Fundamentals & HTML Styling

Understand how CSS interacts with HTML to control layout, font, color, spacing, and structure.

1️⃣ Linking External CSS
<head>
  <link rel="stylesheet" href="styling.css">
</head>
  • 🔹 Function/Class Used: <link> tag
  • 📘 What You Learn: How to apply external CSS styles to HTML pages.
  • ♻️ Reusability Insight: External stylesheets make it easy to update design across pages.
  • 💬 Inline Comments: Ensure the path to the CSS file is correct.
  • 📎 Related Topics: internal style, inline style, href
2️⃣ Styling Elements with Class
<div class="box">Styled Content</div>
  • 🔹 Function/Class Used: class attribute
  • 📘 What You Learn: Applying CSS rules using class selectors.
  • ♻️ Reusability Insight: Classes allow consistent styling across multiple elements.
  • 💬 Inline Comments: Classes should be semantic and reusable.
  • 📎 Related Topics: .class selector, DRY CSS
3️⃣ Using ID Selectors
<div id="header">Site Header</div>
  • 🔹 Function/Class Used: id attribute
  • 📘 What You Learn: Styling unique HTML elements.
  • ♻️ Reusability Insight: IDs are best used for single-use components.
  • 💬 Inline Comments: Avoid reusing the same ID across multiple elements.
  • 📎 Related Topics: #id selector, specificity
4️⃣ Font and Text Styling
<h1 style="font-family: Arial; font-size: 24px;">Hello</h1>
  • 🔹 Function/Class Used: style attribute, font-family
  • 📘 What You Learn: Inline styling to control text appearance.
  • ♻️ Reusability Insight: Use CSS files instead of inline for better maintainability.
  • 💬 Inline Comments: Avoid inline styling in large-scale projects.
  • 📎 Related Topics: text-align, font-size, CSS hierarchy
5️⃣ CSS Box Model Example
<div class="box">
  CENTRAL
</div>
  • 🔹 Function/Class Used: margin, padding, border
  • 📘 What You Learn: Understanding how the box model works.
  • ♻️ Reusability Insight: Essential for layout and spacing.
  • 💬 Inline Comments: Adjust spacing using CSS not inline styles.
  • 📎 Related Topics: display, width, height
6️⃣ Nested Elements
<div>
  Outer Div
  <span>Inline text</span>
</div>
  • 🔹 Function/Class Used: <div>, <span>
  • 📘 What You Learn: Differentiating block vs inline elements.
  • ♻️ Reusability Insight: Helps structure layouts and group styles.
  • 💬 Inline Comments: Use span for inline styling within block containers.
  • 📎 Related Topics: block, inline, semantic HTML
