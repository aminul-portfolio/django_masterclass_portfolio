3️⃣ Using Django Filters and Tags
<h2>{{ first_name|upper }}</h2>
<p>{% lorem %}</p>
<p>Date: {% now "jS F Y" %}</p>
  • 🔹 Function/Class Used: Filters (|) and tags ({% ... %})
  • 📘 What You Learn: Format strings, generate dummy text, insert dates
  • ♻️ Reusability: Used across UI for formatting and development
  • 💬 Inline Comments: Improves display and UX through simple tools
  • 📌 Related Topics: Filters, template tags, now, lorem
4️⃣ Conditional and Looping Logic
<ul>
  {% for item in some_list %}
    {% if item == 2 %}
      <li>Two</li>
    {% else %}
      <li>{{ item }}</li>
    {% endif %}
  {% endfor %}
</ul>
  • 🔹 Function/Class Used: {% for %}, {% if %}
  • 📘 What You Learn: Render lists with conditional logic
  • ♻️ Reusability: Great for menus, listings, and filters
  • 💬 Inline Comments: Combines iteration with logic for dynamic views
  • 📌 Related Topics: Loops, conditionals, dynamic rendering
5️⃣ Custom 404 Page
<html>
              <head><title>Page Not Found</title></head>
              <body>
                <h1>Oops! This page does not exist.</h1>
                <p>Return to the <a href="/">home page</a>.</p>
              </body>
            </html>
  • 🔹 Function/Class Used: HTML template for error handling
  • 📘 What You Learn: Customize 404 error user experience
  • ♻️ Reusability: Helps in branding and user retention
  • 💬 Inline Comments: Friendly fallback for broken URLs
  • 📌 Related Topics: 404 page, custom error templates
6️⃣ Template Linking with {% url %} Tag

<a href={% url 'my_app:home' %}>Home</a>
<a href={% url 'my_app:about' %}>About Us</a>
  • 🔹 Function/Class Used: {% url %}
  • 📘 What You Learn: Generate internal links dynamically using named routes
  • ♻️ Reusability: Keeps links valid even when URL patterns change
  • 💬 Inline Comments: Centralizes routing for better maintainability
  • 📌 Related Topics: Named URLs, reverse resolution, URL dispatcher

  • 📌 Related Topics: Template inheritance, site structure

🔟 Conditional Rendering with Authentication

  <p>Welcome, {{ user.username }}!</p>
{% else %}

{% endif %}
  • 🔹 Function/Class Used: {% if %}
  • 📘 What You Learn: Conditionally show content based on auth status
  • ♻️ Reusability: Used across user dashboards and secure pages
  • 💬 Inline Comments: Implements basic access control in UI
  • 📌 Related Topics: Authentication, role-based access, session state
